# 採点効率化Webアプリケーション 実装計画

採点業務を大幅に効率化するためのWebアプリケーションの実装計画です。いただいたフィードバックをもとに、**ローカルPCへのソフトのインストールが不要で、Webブラウザのみで完結するアーキテクチャ**へ更新しました。

## 概要とシステムアーキテクチャ

すべての処理（画像変換、OCR、ファイル生成）をブラウザ上で行う、**完全フロントエンド（クライアントサイド）完結型**のアプリケーションとします。これにより、サーバー構築やソフトのインストールなしでWeb上で手軽に動かすことができます。

*   **フレームワーク**: React (Vite) + Vanilla CSS
    *   洗練されたモダンなデザインを実装します。
*   **ファイル読み書き**: File System Access API (Chrome/Edge推奨)
    *   ユーザーがブラウザ上でローカルの特定のフォルダを選択すると、ブラウザが直接そのフォルダ内に「images」や「trimmed」フォルダを作成し、画像を書き込めるようにします。
*   **PDF/画像処理**: `pdf.js` (PDFをCanvasに描画してJPEG化)、標準のCanvas API (画像のトリミング処理)
*   **OCRエンジン**: `Tesseract.js` (ブラウザ内で動くWebAssembly版OCR。数字・アルファベット・記号・簡単な単語の認識に使用)
*   **ファイル出力**: `pdf-lib` (採点結果のPDF書き込み)、`exceljs` (Excelファイルの作成)
*   **状態管理**: ブラウザのIndexedDBを使用し、ブラウザを閉じても採点途中から再開できるようにします。

## 機能ごとの実装方針

### 1. 初期設定とフォルダアクセス
*   **フォルダ選択**: ブラウザの File System Access API を使い、作業用フォルダをユーザーに選択させます。
*   **初期化**: 選択されたフォルダ内に `images`, `trimmed`, `Name`, 各問題（`q001`など）のフォルダを作成します。
*   **設定**: 出席番号（開始・終了）、欠席者、模範解答・答案PDFのアップロードを行います。

### 2. PDFからJPEGへの変換
*   **処理**: `pdf.js` を使用し、アップロードされたPDFをブラウザ上でページごとにCanvasにレンダリングし、JPEGとして取得。
*   **保存**: 「出席番号.jpeg」「模範解答.jpeg」として、選択したフォルダ内の `images` ディレクトリに保存します。

### 3. トリミング領域の設定と実行
*   **領域設定UI**: 模範解答画像を表示し、ドラッグ＆ドロップで「名前欄」「各解答欄」の矩形を設定。
*   **トリミング処理**: Canvas APIを用いて一括でトリミングを実行。
*   **保存先**:
    *   名前欄: `trimmed/Name/出席番号_name.jpeg`
    *   各問題: `trimmed/q001/出席番号_001.jpeg`
    *   模範解答: `trimmed/模範解答_001.jpeg` など。

### 4. 配点・自動採点・観点の設定
*   各問題について、配点、部分点の許可、自動採点（OCR）のON/OFF、観点（1〜3）を設定可能にします。

### 5. 採点の実行（UIと機能）
*   **横断型採点UI**: 全生徒の問題一覧をグリッド表示。
*   **★OCR結果の確認と修正**: 自動採点（OCR）をONにした問題でも、結果（正解と判定されたものも含む）を画面上で人間が確認でき、クリックやショートカットキーで判定を修正できるようにします。
*   **★ホバーによる名前確認**: 画像にカーソル（マウス）を乗せた際に、`Name` フォルダのその生徒の名前画像をポップアップ表示し、誰の解答かすぐに分かるようにします。
*   **キーボード操作**: テンキーやショートカットキー（1:正解, 2:不正解, 3:部分点）で操作。
*   **色分け**: 正解（緑）、不正解（赤）、部分点（水色）の半透明オーバーレイをCSSで適用。
*   **一括操作**: 「未入力は全て不正解にする」「一括正解」ボタン。
*   **模範解答ピン留め**: 画面上部にトグル式で模範解答を固定表示。
*   **保存と再開**: IndexedDBに都度状態を保存し、中断・再開に対応。

### 6. 採点結果の出力
*   **PDF出力**: `pdf-lib` を使用。解答用紙の元画像の上に、採点結果（◯、×、△+点数）を解答欄から少し右にずらして描画し、合計点を指定欄に描画してダウンロード。
*   **Excel出力**: `exceljs` を使用。
    *   シート1: 縦に出席番号、横に問題番号の表（◯、×、△）。
    *   シート2: 出席番号と合計点数の一覧表。
