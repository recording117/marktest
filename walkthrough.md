# 採点効率化Webアプリケーション 開発完了報告

ご要望いただいた採点効率化アプリケーションのすべての機能実装が完了しました。
ローカルにソフトをインストールすることなく、**Webブラウザのみで完結する安全かつ高速なシステム**として構築されています。

## ✨ 実装した主要機能

1.  **完全ブラウザ完結 (File System Access API)**
    *   ユーザーのPC内にあるフォルダを直接指定し、ブラウザから直接フォルダ作成や画像の保存を行います。
2.  **PDFの自動JPEG変換**
    *   アップロードした模範解答と答案用紙のPDFを、ページごとに自動で高画質なJPEG画像に変換します。
3.  **ドラッグ＆ドロップでの直感的なトリミング**
    *   模範解答の画像を見ながら、「名前欄」や各「問題の解答欄」をマウスで囲むだけで、すべての生徒の答案を一括でトリミングします。
4.  **Tesseract.js による自動採点 (OCR)**
    *   記号や数字、簡単な単語の問題に対し、自動で文字認識を行い、模範解答と一致していれば自動的に正解（◯）とする機能を搭載しています。
5.  **超高速なキーボード採点UI**
    *   全生徒の解答が1つの画面に並びます。
    *   **ショートカットキー**: `1`で正解、`2`で不正解、`3`で部分点入力。
    *   **ホバー機能**: 誰の解答かわからなくなった時、画像にマウスを乗せるとその生徒の「名前」画像がポップアップで表示されます。
    *   **一括操作**: 「未入力をすべて不正解にする」ボタンで、採点漏れや白紙解答を一瞬で処理できます。
6.  **採点結果のPDF・Excel出力**
    *   **PDF出力**: 答案用紙の元画像に、赤色で大きく「◯」「×」「△（点数）」を描画し、右下に合計点を記した新しいPDFを生成します。
    *   **Excel出力**: 問題別の詳細な結果シートと、合計点数一覧シートを持つExcelファイルを自動生成します。

## 🚀 アプリの使い方

現在、バックグラウンドで開発サーバーが起動しています。以下のURLにアクセスしてご利用ください。

**👉 [http://localhost:5173](http://localhost:5173)**

> [!IMPORTANT]
> **推奨ブラウザ**
> 本システムは高度なローカルファイルアクセス技術を利用しているため、必ず **Google Chrome** または **Microsoft Edge** のPC版でアクセスしてください。

### ご利用のステップ

1.  **初期設定**: 作業用の空フォルダをPC上に作成し、画面からそのフォルダを選択します。その後、出席番号を設定し、PDFをアップロードします。
2.  **PDF変換**: ボタンを押して画像を生成します。
3.  **トリミング**: 画面左側のメニューから「名前欄」を選んで枠を書き、次に「問題001」を選んで枠を書きます。終わったら「全画像をトリミング」を実行します。
4.  **配点設定**: 問題ごとの配点や、OCRのON/OFFを設定します。
5.  **採点実行**: 画像をクリックしてフォーカスを合わせ、キーボードの `1`, `2` などを押してサクサク採点します。
6.  **結果出力**: PDFとExcelを出力して完了です。
