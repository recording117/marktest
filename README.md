# 採点支援Webアプリケーション（名称未設定） 使い方マニュアル

このアプリは、テストの解答用紙（PDF形式）をブラウザ上で読み込み、画像の自動切り出しやOCR（文字認識）による自動採点、そして手動採点をスムーズに行うためのツールです。
先生方の採点業務の負担を劇的に減らすことを目的としています。

---

## 🌟 主な特徴
1. **ブラウザ完結で安全**: インターネット上のサーバーに生徒の解答データやPDFを送信しません。すべてお使いのPC内で処理されます。
2. **自動切り出し機能**: 1枚のPDFから、生徒ごとの解答用紙を自動で画像として切り出します。
3. **AI (OCR) による自動採点**: 記号や数字などの簡単な解答を自動で読み取り、正誤判定を行います。
4. **サクサク動く手動採点**: キーボード操作だけで、次々と解答を採点していくことができます。
5. **Excel・PDF出力**: 採点結果を自動集計し、Excelや成績返却用のPDFとして出力できます。

---

## 🚀 準備するもの
*   **模範解答のPDF**（1ページのみ）
*   **生徒の解答用紙のPDF**（生徒全員分がスキャンされた1つのPDFファイル）
    *   ※ 出席番号順にスキャンされている必要があります。
*   Google Chrome または Microsoft Edge ブラウザ（最新版）

---

## 📝 基本的な使い方（ステップ・バイ・ステップ）

左側のメニュー（1〜6）の順番に従って進めていきます。

### 1. 初期設定
1. **作業フォルダの選択**: 「フォルダを選択」ボタンを押し、PC内の空のフォルダを指定します。このフォルダに画像やデータが一時保存されます。（※ブラウザからフォルダへのアクセス許可を求められたら「許可」してください）
2. **出席番号の設定**:
    *   **対象の出席番号**: `1-40` のように入力します。複数クラスに解答用紙がまたがっている場合は `3101-3126, 3201-3225` のようにカンマとハイフンで指定できます。
    *   **欠席者**: 欠席した生徒の番号をカンマ区切りで入力します（例: `3, 12`）。
3. **PDFファイルの登録**: 模範解答のPDFと、生徒の解答用紙のPDFをそれぞれアップロードします。

### 2. PDF読み込み・画像生成
*   「PDFから画像を生成する」ボタンを押します。
*   アップロードしたPDFが、生徒一人一人の画像（JPEG）に変換・保存されます。

### 3. 解答欄（トリミング領域）の設定
1. 左側の画面で模範解答の画像を見ながら、解答欄をマウスで**ドラッグして枠で囲みます**。
2. 右側に枠の一覧が表示されます。「名前欄」として設定したり、「問1」「問2」と名前を変更できます。
3. （必要に応じて、総合得点を記入する枠なども設定できます）
4. 全ての枠を設定し終えたら「画像の切り出しを実行」ボタンを押します。

### 4. 配点・自動採点（OCR）設定
1. 設問ごとに**配点**や**観点（1〜3）**を設定します。
2. **自動採点(OCR)** を行う問題はチェックを入れます。
3. **【重要】正解（模範解答）の入力**:
   * 「模範解答を画像から読み込む」を押すと、正解の文字が自動入力されます。
   * もし間違って読み込まれた場合は、直接テキストボックスをクリックして修正してください。（数字のみ、カタカナのみ等、ここで設定した文字種に合わせてOCRの精度が最適化されます）
4. 「OCRで生徒の解答を自動採点する」ボタンを押すと、自動的にマルバツが判定されます。*OCRの精度はあまり高くありません。そのため、自動採点後は必ず見直しを行いましょう。*

### 5. 採点・結果確認
*   画面上に、生徒ごとの解答画像が並んで表示されます。
*   キーボード操作でサクサク採点できます：
    *   `1` キー : **正解（◯）**
    *   `2` キー : **不正解（×）**
    *   `3` キー : **部分点（△）**
    *   `Enter` キー : 次の設問へ進む
    *   `Backspace` キー : 前の設問へ戻る
    *   `矢印（↓・→）` キー : 次の生徒へ進む
*   上のタブで「正解のみ」「不正解のみ」などに絞り込んで、まとめて確認することも可能です。

### 6. 結果の出力
*   **Excel形式で出力**: 設問ごとの正誤や、観点別の合計点数がまとまったExcelファイル（採点結果.xlsx）がダウンロードできます。
*   **採点済みPDFを出力**: 生徒の解答用紙の画像の上に、赤いマルやバツ、点数が印字された状態の新しいPDFが生成されます。これを印刷して生徒に返却できます。

## 💡 トラブルシューティング
*   **Q. 作業フォルダを選択できない / エラーが出る**
    *   A. ブラウザのセキュリティ設定で、ローカルファイルへのアクセスが許可されていない可能性があります。URLバーの左側にある「鍵マーク（または設定マーク）」から、ファイルアクセスの権限を確認してください。
*   **Q. OCR（自動採点）が全然正解しない**
    *   A. 手順4の表の「正解（模範解答）」欄が正しく入力されているか確認してください。ここに入力された文字（数字、カタカナなど）をヒントにしてAIが読み取りを行います。
*   **Q. 途中で作業をやめたい**
    *   A. 各画面で設定した内容は自動的に保存（キャッシュ）されています。次回同じ作業フォルダを選択すれば、途中から再開できます。
